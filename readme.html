Importing XtendM3 Table to M3

Guide to import the XtendM3 table to M3



Instructions

  1. Download the XtendM3 table export file.
  2. Open the file and copy it's content.
  3. Open M3 in the browser.
  4. Open the browser's console (Ctrl + Shift + J).
  5. Switch the context of the console by clicking the dropdown and selecting m3h5_*

  6. Paste the text in the console and press Enter.

The table should now be imported into M3.
